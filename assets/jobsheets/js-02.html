A. Tujuan Pembelajaran

Setelah menyelesaikan jobsheet ini, peserta didik mampu:

  1. Memahami prinsip kerja motor induksi tiga fasa dan metode pengendaliannya.
  2. Membuat program Ladder Diagram (LD) untuk start/stop motor tiga fasa menggunakan PLC.
  3. Mengimplementasikan kontrol Direct-On-Line (DOL) dan Star-Delta melalui PLC Siemens S7-1200.
  4. Menerapkan proteksi overload dan interlock pada program kontrol motor.

B. Dasar Teori

Motor tiga fasa merupakan aktuator utama dalam industri manufaktur. Pengendalian motor melalui PLC memberikan keunggulan berupa fleksibilitas program, kemudahan integrasi proteksi, dan kemampuan monitoring real-time.

Metode pengendalian motor tiga fasa:

  • Direct-On-Line (DOL) – Motor langsung dihubungkan ke sumber tegangan penuh. Arus starting bisa mencapai 5–7× arus nominal.
  • Star-Delta – Motor dihidupkan dalam konfigurasi bintang (tegangan berkurang) lalu beralih ke segitiga (tegangan penuh) setelah beberapa detik. Arus starting lebih rendah ±1/3 dari DOL.
  • Variable Frequency Drive (VFD) – Frekuensi suplai diatur untuk mengontrol kecepatan motor secara halus.
💡 Catatan Penting Pada PLC S7-1200, output DO (Digital Output) mengendalikan koil kontaktor. Kontaktor inilah yang menyambung/memutus arus motor. PLC tidak langsung mengalirkan arus motor.

D. Peta I/O (Address PLC)

Tag Address Tipe Fungsi
START I0.0 Input DI Tombol Start (NO) – aktif saat ditekan
STOP I0.1 Input DI Tombol Stop (NC) – aktif normal
OVERLOAD I0.2 Input DI Kontak TOR – trip = FALSE
KM_MAIN Q0.0 Output DO Kontaktor utama motor
KM_STAR Q0.1 Output DO Kontaktor konfigurasi bintang (Star)
KM_DELTA Q0.2 Output DO Kontaktor konfigurasi segitiga (Delta)
LAMP_RUN Q0.3 Output DO Lampu indikator motor berjalan
LAMP_FAULT Q0.4 Output DO Lampu indikator gangguan/overload

E. Program Ladder Diagram

Network 1 – Kontrol DOL (Direct-On-Line):

|--[I0.0 START]--+--[Q0.0 KM_MAIN]--| (Q0.0 KM_MAIN)--| | | | | [Q0.0 KM_MAIN (self-hold)] | | | |--[I0.1 STOP /NC]--[I0.2 OVL /NC]--[Q0.0]-----|

Network 2 – Star-Delta Timer (Star aktif 5 detik, lalu beralih ke Delta):

N2: KM_MAIN ON → KM_STAR ON |--[Q0.0]--[/Q0.2 interlock Delta]--| (Q0.1 KM_STAR) N3: Timer TON – tunda 5 detik |--[Q0.0]--| TON T1 (PT=5s) |--[T1.Q]--| (Q0.2 KM_DELTA) N4: Matikan Star saat Delta aktif |--[T1.Q]--| (RESET Q0.1 KM_STAR)
⚠️ INTERLOCK WAJIB Pastikan KM_STAR dan KM_DELTA tidak bisa aktif bersamaan! Tambahkan kontak NC dari Q0.1 pada rung Q0.2 dan sebaliknya, baik secara hardware (kontak NC kontaktor) maupun software (ladder interlock).

F. Langkah Kerja

  1. Buat project baru di TIA Portal V17, pilih CPU Siemens S7-1214C DC/DC/DC.
  2. Konfigurasi alamat I/O sesuai tabel peta I/O pada bagian D di atas.
  3. Buat program Ladder di OB1 sesuai skema Network 1–4 pada bagian E.
  4. Lakukan simulasi menggunakan PLC-SIM sebelum diunduh ke hardware.
  5. Rangkai wiring fisik pada trainer sesuai diagram rangkaian (minta guru untuk verifikasi sebelum energize).
  6. Download program ke PLC, lakukan pengujian satu per satu:
    • Tekan START → Motor harus menyala dalam mode Star (5 detik).
    • Setelah 5 detik → KM_STAR off, KM_DELTA on (motor dalam mode Delta).
    • Tekan STOP → semua kontaktor off, motor berhenti.
    • Simulasikan trip TOR → LAMP_FAULT harus menyala, motor off.
  7. Dokumentasikan hasil dengan foto dan isi tabel pengamatan di bagian G.

G. Tabel Pengamatan

No Kondisi Uji KM_MAIN KM_STAR KM_DELTA Hasil

H. Pertanyaan Evaluasi

  1. Mengapa arus starting motor DOL bisa 5–7× lebih besar dari arus nominal? Jelaskan!
  2. Apa fungsi Thermal Overload Relay dalam sistem kontrol motor? Bagaimana cara kerjanya?
  3. Jelaskan mengapa interlock antara KM_STAR dan KM_DELTA sangat penting dan apa akibatnya jika tidak dipasang!
  4. Berapa nilai PT Timer yang dibutuhkan jika ingin motor beralih dari Star ke Delta dalam 8 detik?
  5. Bagaimana caranya agar motor bisa dijalankan dari dua lokasi berbeda (tombol lokal + tombol remote)?
