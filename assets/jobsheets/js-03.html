A. Tujuan Pembelajaran

  1. Memahami jenis-jenis Timer IEC (TON, TOF, TP) pada PLC Siemens S7-1200.
  2. Memahami jenis-jenis Counter IEC (CTU, CTD, CTUD) dan cara penggunaannya.
  3. Membuat program Ladder Diagram yang menggunakan Timer dan Counter untuk kontrol otomatis.
  4. Mengimplementasikan skenario nyata: lampu berkedip otomatis, penghitung produk, dan kontrol tunda.

B. Dasar Teori Timer

Timer pada PLC digunakan untuk membuat penundaan waktu (delay) dalam eksekusi program. Tiga jenis timer IEC yang umum digunakan:

⏱ TON – Timer ON Delay

Output aktif setelah IN aktif selama waktu PT. Saat IN mati, output langsung mati. Digunakan untuk penundaan start.

⏱ TOF – Timer OFF Delay

Output tetap aktif selama PT setelah IN dimatikan. Digunakan untuk penundaan stop (misal: pendingin kipas).

⏱ TP – Pulse Timer

Output aktif selama tepat PT detik setiap kali IN naik (rising edge). Berguna untuk membuat pulsa dengan durasi pasti.

C. Dasar Teori Counter

Counter menghitung jumlah event (pulsa rising edge). Tiga jenis counter IEC:

🔢 CTU – Count Up

Menghitung naik dari 0 hingga PV. Output Q aktif saat CV ≥ PV. Input R (Reset) mengembalikan CV ke 0.

🔢 CTD – Count Down

Menghitung turun dari PV menuju 0. Output Q aktif saat CV ≤ 0. Input LD (Load) mengisi ulang nilai PV.

🔢 CTUD – Up/Down Counter

Menghitung naik (CU) dan turun (CD). QU aktif saat CV ≥ PV, QD aktif saat CV ≤ 0. Fleksibel untuk berbagai aplikasi.

💡 Data Type Timer & Counter Di TIA Portal (S7-1200), blok Timer dan Counter bertipe IEC_TIMER dan IEC_COUNTER. Blok ini membutuhkan instance DB. Buat DB tersendiri atau gunakan multi-instance DB agar lebih efisien.

D. Peta I/O (Address PLC)

Tag Address Tipe Fungsi
BTN_START I0.0 Input DI Tombol mulai hitungan / aktifkan timer
BTN_RESET I0.1 Input DI Reset counter dan timer
SENSOR_PRODUK I0.2 Input DI Sensor proximity penghitung produk
LAMP_1 Q0.0 Output DO Lampu indikator delay ON (TON)
LAMP_2 Q0.1 Output DO Lampu berkedip (TP)
BUZZER Q0.2 Output DO Buzzer aktif saat counter penuh
KONVEYOR Q0.3 Output DO Motor konveyor berjalan

E. Program Ladder Diagram

Network 1 – TON Delay 3 detik (Konveyor ON setelah 3 detik):

|--[I0.0 BTN_START]--| TON "Timer1" |--[Timer1.Q]--| (Q0.3 KONVEYOR) IN Q PT: T#3s ET: (elapsed)

Network 2 – TP Pulse Lamp (Lampu berkedip 0.5 detik setiap siklus):

|--[I0.0]--[/Timer2.Q]--| TP "Timer2" | IN Q-----(Q0.1 LAMP_2) PT: T#500ms

Network 3 – CTU Counter Produk (Alarm setelah 10 produk terdeteksi):

|--[I0.2 SENSOR]--| CTU "Counter1" |--[Counter1.Q]--| (Q0.2 BUZZER) CU Q R: I0.1 RESET PV: 10 CV: (current)

Network 4 – Reset otomatis Counter setelah BUZZER 2 detik:

|--[Counter1.Q]--| TON "Timer3" |--[Timer3.Q]--| (RESET Counter1.R) IN Q PT: T#2s

F. Langkah Kerja

  1. Buat project baru di TIA Portal, tambahkan CPU S7-1214C.
  2. Buat global DB untuk instance Timer dan Counter (atau gunakan local tag di OB1).
  3. Masukkan program Ladder pada OB1 sesuai Network 1–4 di atas.
  4. Pastikan resolusi timer sudah sesuai (default S7-1200: 1ms untuk DB timer).
  5. Jalankan simulasi di PLC-SIM, gunakan Watch Table untuk memantau nilai ET (elapsed time) dan CV (current value).
  6. Uji skenario berikut dan catat hasilnya pada tabel pengamatan:
    • Tekan START → konveyor harus menyala setelah tepat 3 detik.
    • Lampu LAMP_2 harus berkedip selama START aktif (periode ±1 detik).
    • Berikan 10 pulsa pada SENSOR_PRODUK → BUZZER harus aktif 2 detik lalu reset.
  7. Implementasikan pada hardware trainer, verifikasi dengan stopwatch untuk akurasi timer.

H. Pertanyaan Evaluasi

  1. Apa perbedaan utama antara TON dan TOF? Berikan contoh aplikasinya di industri!
  2. Jelaskan apa yang terjadi jika input IN pada TON dimatikan sebelum elapsed time (ET) mencapai preset time (PT)!
  3. Bagaimana cara membuat lampu yang berkedip (ON 1 detik, OFF 1 detik) secara terus-menerus menggunakan Timer? Jelaskan prinsipnya!
  4. Jika PV Counter = 50 dan sensor mengirim 60 pulsa, berapa nilai CV akhir? Apakah output Q tetap aktif?
  5. Pada skenario konveyor bottling plant (pengisian botol), Timer dan Counter jenis apa yang paling cocok digunakan? Jelaskan alasanmu!
