🏭 MODUL 06 – MENENGAH

Kontrol Konveyor & Motor

Program PLC untuk sistem konveyor belt: start/stop, forward/reverse, kontrol kecepatan dengan inverter, dan sekuensial multi-konveyor.

Kelas: XII TEI & TPM
Durasi: 3 × 45 menit
Level: Menengah
Topik: 9 Sub-Topik

1. Tujuan Pembelajaran

  1. Menjelaskan komponen-komponen sistem konveyor belt industrial dan fungsinya.
  2. Merancang program PLC untuk kontrol start/stop konveyor dengan self-holding.
  3. Mengimplementasikan kontrol forward/reverse konveyor dengan interlock penuh.
  4. Memahami cara mengendalikan kecepatan motor dengan Variable Speed Drive (VSD/Inverter) melalui output analog PLC.
  5. Merancang sistem sekuensial untuk tiga konveyor (jalur produksi bertahap).
  6. Mengintegrasikan sensor posisi (photoelectric/proximity) untuk otomasi penuh.
  7. Menerapkan sistem proteksi motor: overload relay, thermal relay, dan E-Stop.

2. Sistem Konveyor Belt – Overview

Sistem konveyor belt adalah salah satu aplikasi PLC paling umum di industri manufaktur. Konveyor digunakan untuk memindahkan material atau produk antar stasiun kerja, mesin produksi, atau area gudang secara otomatis dan kontinu.

Motor Listrik 3 Fasa

Penggerak utama konveyor. Tipe: induksi asinkron. Dikontrol PLC melalui kontaktor atau inverter (VSD). Daya: 0.37 kW – 15 kW tergantung beban.

Kontaktor & Thermal Relay

Kontaktor (KM): switch daya utama dikendalikan coil 24VDC. Thermal Relay (OL): proteksi motor dari overload, kontak NC ke PLC untuk sinyal fault.

VSD / Inverter

Variable Speed Drive: mengatur kecepatan motor secara halus (0–100%). PLC mengirim referensi kecepatan via sinyal analog 0–10V (AO) atau 4–20mA.

Sensor & Detektor

Photoelectric/proximity sensor: mendeteksi kehadiran produk di konveyor. Encoder: mengukur kecepatan/posisi belt. Limit switch: mendeteksi posisi akhir.

5. Kontrol Kecepatan dengan Inverter

Inverter (VSD) menerima referensi kecepatan dari output analog PLC. Siemens S7-1200 menggunakan nilai integer 0–27648 untuk mewakili 0–10V (atau 0–20mA). Kecepatan motor diatur proporsional terhadap nilai ini.

💡 Rumus Konversi Kecepatan
  • 100% kecepatan (50 Hz) = 27648 (nilai raw AO)
  • 50% kecepatan (25 Hz) = 13824
  • Kecepatan yang diinginkan (%) × 27648 / 100 = Nilai raw AO
  • Gunakan instruksi NORM_X dan SCALE_X di TIA Portal untuk konversi yang bersih.

6. Sistem Sekuensial Tiga Konveyor

Dalam lini produksi, konveyor sering disusun seri. Urutan start/stop harus terurut untuk mencegah penumpukan produk:

  • Start Sequence: Konveyor paling akhir (Conv-3) START lebih dulu, lalu Conv-2, lalu Conv-1. Ini mencegah produk menumpuk di awal jalur.
  • Stop Sequence: Conv-1 STOP dulu (tidak ada produk baru), lalu Conv-2 setelah delay, lalu Conv-3 setelah Conv-2 kosong.
Start Sequence (menggunakan Timer delay 3 detik antar konveyor): Network 1 – Conv-3 start langsung: |--[ I0.0 START ]--+--[/ I0.1 STOP]--[/ I0.4 ESTOP]--( Q0.2 MOTOR_CONV3 )--| |--[ Q0.2 HOLD3 ]--+ Network 2 – Conv-2 start setelah delay 3 detik dari Conv-3: |--[ Q0.2 MOTOR_CONV3 ]--|TON T1: T#3s|--( Q0.1 MOTOR_CONV2 )--| Network 3 – Conv-1 start setelah delay 3 detik dari Conv-2: |--[ Q0.1 MOTOR_CONV2 ]--|TON T2: T#3s|--( Q0.0 MOTOR_CONV1 )--|

7. Proteksi Motor – Fault Handling

  • Thermal Relay (OL Relay): Memutus kontaktor saat motor overload (panas berlebih). Kontak NC-nya dipasang ke input PLC untuk mematikan semua output dan menyalakan alarm FAULT.
  • Undercurrent Detection: Jika motor berputar tanpa beban (belt putus), arus motor akan turun drastis. PLC bisa mendeteksi ini via sensor arus dan alarm "Belt Putus".
  • Jam Detection: Jika ada objek menyangkut, kecepatan encoder turun drastis. PLC membandingkan speed setpoint vs actual dan alarm jika deviasi > 20%.
