🔌 MODUL 03 – DASAR

Wiring & Instalasi Input/Output PLC

Teknik instalasi kabel sensor ke terminal input dan output ke aktuator secara aman dan benar sesuai standar industri.

Kelas: XI/XII TEI & TPM
Durasi: 3 × 45 menit
Level: Dasar
Topik: 7 Sub-Topik

1. Tujuan Pembelajaran

  1. Mengidentifikasi jenis-jenis sensor input digital dan analog beserta karakteristiknya.
  2. Memahami konfigurasi input Sinking (NPN) dan Sourcing (PNP) pada PLC.
  3. Memasang kabel sensor proximity, push button, dan limit switch ke terminal input PLC.
  4. Memahami jenis output relay dan transistor pada modul output PLC.
  5. Memasang kabel aktuator (motor starter, lampu, solenoid) ke terminal output PLC.
  6. Menerapkan standar pengkabelan: warna kabel, labeling, dan proteksi.
  7. Melakukan pengujian dan verifikasi wiring menggunakan multimeter.
🔴 KESELAMATAN KERJA UTAMA SELALU matikan sumber tegangan (MCB ke OFF) sebelum memasang atau melepas kabel apapun pada sistem PLC. Bekerja pada tegangan >50V AC atau >120V DC memerlukan APD lengkap (sarung tangan isolasi, sepatu safety, kacamata). Jangan pernah menyentuh terminal output saat PLC dalam kondisi RUN.

2. Jenis-Jenis Input Digital

Push Button (Tombol Tekan)

Input mekanis paling sederhana. Tipe NO: kontak menutup saat ditekan. Tipe NC: kontak membuka saat ditekan. Contoh: tombol START/STOP, tombol emergensi.

Limit Switch

Sensor posisi mekanis. Diaktifkan saat tuas atau roller fisik tersentuh objek. Digunakan untuk mendeteksi posisi akhir (end of travel) konveyor, pintu, atau aktuator pneumatik.

Proximity Sensor (NPN/PNP)

Mendeteksi kehadiran objek logam tanpa kontak fisik. Induktif: hanya logam. Kapasitif: plastik, cairan, non-logam juga. Output: NPN (sinking) atau PNP (sourcing).

Photoelectric Sensor

Mendeteksi benda menggunakan cahaya inframerah. Tipe Thru-beam: lebih akurat, butuh pemancar dan penerima terpisah. Tipe Reflective: lebih praktis, satu unit. Digunakan di konveyor sorting.

3. Konfigurasi NPN vs PNP (Sinking vs Sourcing)

Ini adalah konsep yang paling sering membingungkan pemula. Perbedaan utama ada pada polaritas sinyal ke terminal input PLC:

Aspek Sensor NPN (Sinking) Sensor PNP (Sourcing)
Sinyal Output Sensor Menarik arus ke GND (0V) saat aktif Mendorong arus dari +24V saat aktif
Input PLC yang cocok Input Sourcing (common = +24V) Input Sinking (common = 0V/GND)
Tegangan terminal input saat aktif ~0V (ditarik ke GND oleh sensor) ~24V (didorong oleh sensor)
Prevalensi Umum di Asia (Jepang, Korea) Umum di Eropa & Amerika
Siemens S7-1200 Butuh modul input khusus NPN / jumper Default input S7-1200 = PNP (sourcing)
⚠️ Hati-Hati Perbedaan NPN/PNP Memasang sensor NPN ke input yang dikonfigurasi PNP (atau sebaliknya) bisa menyebabkan input "selalu ON" atau tidak pernah aktif. Selalu periksa datasheet sensor dan manual modul input PLC sebelum wiring.

4. Standar Wiring dan Pengkabelan

Jenis Kabel Kode Warna Standar (IEC) Penggunaan
Fasa (L) Coklat / Hitam / Abu-abu Tegangan AC dari sumber daya
Netral (N) Biru Netral AC
Grounding (PE) Hijau-Kuning Grounding proteksi, WAJIB
DC +24V Merah Suplai positif sensor/aktuator DC
DC 0V/GND Hitam / Biru gelap Ground negatif DC
Sinyal Input Oranye / Kuning Kabel sinyal dari sensor ke terminal input
Sinyal Output Ungu / Putih Kabel sinyal dari terminal output ke aktuator
  • Gunakan kabel flexible (serabut) untuk wiring panel, bukan kabel solid (NYA).
  • Pasang ferrule (end sleeve) di setiap ujung kabel yang dimasukkan ke terminal.
  • Gunakan cable duct (trunking) untuk merapikan kabel dan memudahkan perawatan.
  • Beri label/marking pada setiap kabel sesuai nomor terminal (contoh: I0.0, Q0.1, dll.).
  • Pisahkan kabel daya (220V) dari kabel sinyal untuk mengurangi noise elektromagnetik.

5. Jenis Output PLC dan Beban yang Cocok

Output Relay (R)

Berupa kontak relay mekanis. Bisa mengendalikan AC maupun DC. Arus maks ~2A per output. Cocok untuk: lampu, solenoid, dan motor kecil. Kekurangan: lambat (10ms), kontak aus.

Output Transistor (T)

Solid-state, hanya untuk beban DC. Sangat cepat (<1ms). Cocok untuk: sinyal ke inverter, output frekuensi tinggi (pulse train), indikator LED. Maks ~0.5A per kanal.

💡 Menggunakan Relay Eksternal (Interposing Relay) Untuk beban induktif besar (kontaktor motor 3 fasa), selalu gunakan relay eksternal (interposing relay) sebagai buffer antara output PLC dan kontaktor. Ini melindungi modul output PLC dari lonjakan arus induktif dan memperpanjang umur modul.

6. Prosedur Pengujian Wiring (Commissioning)

  1. Visual Check: Periksa semua terminal sudah terpasang kuat, tidak ada kabel longgar atau bersentuhan.
  2. Continuity Test: Gunakan multimeter (mode Ω atau beeper) untuk verifikasi jalur kabel sambung benar.
  3. Insulation Test: Pastikan tidak ada short circuit (hubungan singkat) antara kabel daya dan sinyal.
  4. Power ON Test (Low Voltage): Nyalakan tegangan DC 24V saja, periksa lampu indikator status input/output di CPU.
  5. Force Output Test: Gunakan fitur "Force" di TIA Portal untuk mengaktifkan output secara manual dan verifikasi aktuator bergerak sesuai wiring diagram.
  6. Input Verification: Aktifkan setiap sensor secara manual dan verifikasi LED input di modul PLC menyala sesuai.

7. Keselamatan Listrik dalam Wiring PLC

  • Selalu pasang MCB (Miniature Circuit Breaker) pada setiap jalur daya sebelum masuk PLC.
  • Grounding sistem harus terhubung ke PE bus panel, lalu ke electroda ground gedung.
  • Gunakan surge protector (SPD) pada jaringan listrik untuk melindungi PLC dari lonjakan tegangan petir.
  • Kabel sinyal analog (0–10V, 4–20mA) harus menggunakan kabel shielded (berpelindung) untuk menghindari noise.
  • Shield (pelindung) kabel analog dihubungkan ke ground hanya di satu sisi (biasanya sisi panel PLC).
