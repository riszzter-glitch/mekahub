Mulai Gratis

🛠️ Panduan Menggunakan Meka-Lab

1

Pilih Skenario

Klik dropdown "Skenario" dan pilih tingkat latihan: Dasar, Menengah, Lanjut, atau Expert. Klik Load Template 📋 untuk memuat kode contoh.

2

Susun Kode PLC

Gunakan toolbar NO / NC / COIL / TON / CTU untuk menyisipkan instruksi, atau ketik langsung di area kode. Klik Sync 🔄 untuk menyinkronkan diagram.

3

Jalankan & Amati

Tekan ▶ Jalankan Simulasi. Perhatikan Smart Factory View dan pantau I/O Monitor. Gunakan Manual Override untuk mengubah input secara langsung.

💡 Tips: Jika muncul SYSTEM FAULT!, baca pesan error di bawahnya dan periksa logika interlock-mu. Gunakan tombol ⏭ Step untuk mengeksekusi satu siklus PLC dalam mode debug. Tekan ↺ Reset kapan saja untuk memulai ulang dari awal.
